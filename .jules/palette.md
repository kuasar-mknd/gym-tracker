## 2024-03-20 - Global Haptics & Interaction Directives
**Learning:** Discovered a critical bug in the global haptics utility (`useHaptics.js`) where an undefined variable caused the Vibration API to fail. Restoring this utility enabled haptic feedback app-wide. Additionally, identified that using the `v-press` directive is superior to manual Tailwind `active:scale-x` classes as it provides a centralized, consistent interaction model (scale + haptics) that aligns with the "Liquid Glass" design system.
**Action:** Always favor the `v-press` directive for primary interactive elements. Ensure the haptics utility is kept up-to-date with semantic patterns (like `selection` for gestures) to maintain a rich tactile experience.
