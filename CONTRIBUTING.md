# Contributing to Gym Tracker

Thank you for your interest in contributing to Gym Tracker!

## How to Contribute

1. **Fork the repository** and create your branch from `main`.
2. **Setup your environment** (see README for installation).
3. **Make your changes**.
4. **Ensure tests pass** running `php artisan test`.
5. **Format your code** using `./vendor/bin/pint`.
6. **Submit a Pull Request**.

## Pull Request Guidelines

- Provide a clear description of the changes.
- Link any related issues.
- Ensure the CI checks pass.

## Code of Conduct

Please be respectful and follow the standard open-source code of conduct.
